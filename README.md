# A-Novel-RNA-Aptamer-as-Synthetic-Inducer-of-NagR-Controlled-Transcription
A custom Python-based analysis pipeline was developed for extraction and downstream analysis of aptamer sequences from the high-throughput sequencing data. Raw paired-end FASTQ files were first merged and adapter-trimmed using PEAR v0.9.11 with default parameters. Sequence counts were then obtained from the processed reads, and low-abundance sequences were filtered before downstream analysis. In the next step, aptamer-containing reads were identified using custom scripts based on recognition of the constant flanking regions, and reverse complementation was applied when necessary. The extracted aptamer sequences were subsequently assigned to their corresponding aptamer class and SELEX round according to the integrated barcode sequences. Only sequences passing quality control criteria, including expected aptamer length and valid flanking sequence architecture, were retained for further analysis.
For each round, the occurrence of every unique aptamer sequence was counted, and sequence abundance was normalized as reads per million (RPM). To identify motif-containing sequences, extracted NagR aptamer reads were screened against reference motif sequences, including the Nap_1 motif and other motif classes. Motif detection was performed by first searching for exact substring matches and, if no exact match was found, by applying a custom sliding-window mismatch-tolerant search. In this procedure, candidate motif matches were required to satisfy a seed-matching criterion and were accepted only when the total number of mismatches across the full motif did not exceed two. Based on these analyses, summary statistics were generated for each SELEX round, including the total number of matched sequences, RPM values, motif starting-position distributions, numbers of mismatched variants, and nucleotide substitutions at individual motif positions.
